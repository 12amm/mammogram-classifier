# 🩺 Mammogram Mass Classifier

A machine learning project to predict whether a mammogram mass is **Benign** or **Malignant**.

## 📊 Dataset
- UCI Mammographic Mass Dataset
- 961 instances, 5 features

## 🤖 Models Used
| Model | Accuracy |
|-------|----------|
| Neural Network | 82.53% |
| SVM (linear) | 80.00% |
| Logistic Regression | 80.00% |
| KNN | 79.76% |
| Random Forest | 77.47% |
| Naive Bayes | 76.99% |
| Decision Tree | 73.61% |

## 🚀 Streamlit App
Interactive web app to predict mass severity based on patient data.

## 🛠️ Technologies
- Python, Scikit-learn, TensorFlow, Streamlit, Pandas, NumPy
